# GSI_FS_Practice_Assignment1-
Assignment 1: Testcase study 
📌 Overview
This repository contains test cases designed for the Guru99 Bank Demo Site, specifically focusing on the “Add New Customer” functionality. The test cases are structured to validate both positive workflows (successful customer creation) and negative workflows (error handling when invalid data is entered).


The format follows a standard QA test case template with fields such as:

* Test Case ID

* Test Scenario

* Test Steps

* Tested Environment

* Test Data

* Actual Output

* Expected Output

* Remarks
